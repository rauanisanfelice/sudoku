## Instruções

1. Virtual env;
2. Dependências;
3. Enjoy;

### Virtual env
```
virtualenv -p python3 env
source env/bin/activate
```

### Dependências
```
pip3 install -r requirements.txt
```

### Enjoy
```
python3 game.py
```
